PANEL 1 — CUSTOMER KIOSK
Device: Tablet (in-restaurant)   |   Auth: None — Public
EntryScreen  —  Full-screen Dine-In / Takeaway selection. Background slideshow + large touch buttons.
MenuScreen  —  2-panel layout: CategorySidebar (left) + ProductGrid (right). AllergenFilterBar on top. CartFloatingButton.
ProductDetailScreen  —  Hero image, IngredientToggleList (defaults removable), AddonList, QuantitySelector, RecommendedItems overlay.
BeverageSelectionScreen  —  Post-item beverage prompt with BeverageGrid + SkipButton.
CartScreen  —  CartItemRow x N, CartSummary (subtotal/tax/total PKR), EditItemButton, ConfirmOrderButton.
PaymentScreen  —  CashInstructions (go to counter) or CardTerminalPrompt (tap/insert). Auto-detects method.
OrderConfirmScreen  —  OrderNumberDisplay (large #0042), ETADisplay, ThankYouAnimation (Lottie). Auto-reset timer 90s.
IdleResetTimer  —  Global — resets session to EntryScreen after 90 seconds of inactivity.

PANEL 2 — RECEPTIONIST
Device: Desktop / Tablet   |   Auth: Staff ID + Password
ShiftOpeningModal  —  Mandatory on first login — Opening float input. Feeds into cash reconciliation.
OrderQueuePage  —  Live realtime order list. OrderCard shows #, type badge, source badge, items, total, status, time elapsed.
OrderDetailDrawer  —  Slide-in panel. Full order + MarkPaidButton, ModifyOrderButton (2-min window), CancelOrderButton.
TableManagementPage  —  FloorMapCanvas (visual drag-drop). TableNode colour-coded: green=available, red=occupied, yellow=reserved.
ManualOrderPage  —  Receptionist enters orders manually. Full menu navigation + CustomerInfoForm for delivery orders.
CashDrawerPage  —  DrawerBalanceDisplay, TransactionForm (cash-in/out + reason), TransactionHistory, ShiftCloseModal.
SalesDashboardPage  —  Today's live stats: Dine-In / Takeaway / Online / Voice / Grand Total cards.
ModifyOrderModal  —  Edit items/qty/customisations. After 2-min window: requires ManagerApprovalRequired badge.

PANEL 3 — KITCHEN DISPLAY (KDS)
Device: Tablet (wall-mounted, always-on)   |   Auth: Staff ID + Password
KitchenDisplayPage  —  Single always-on screen. Dark high-contrast theme. Large font readable from 2 metres.
KitchenOrderCard  —  Order#, type badge, Voice Order badge if applicable, ElapsedTimerBadge (red >15min), ItemCheckList.
KitchenItemRow  —  Checkbox to mark individual items ready. Customisations highlighted in colour.
MarkOrderReadyButton  —  Full-width CTA. Triggers notification to Receptionist panel and Order Bumper display.
ItemAvailabilityPanel  —  Side panel — StockToggleRow per item. Toggle = instant update to kiosk, website, manager alert.
NewOrderAlert  —  Audio beep + screen flash on new order. AudioAlert component with volume control.
FullscreenGuard  —  Auto re-enters fullscreen if kitchen staff accidentally exits. Prevents panel from being minimised.

PANEL 4 — MANAGER
Device: Desktop / Tablet   |   Auth: Password + PIN (2-step)
DashboardPage  —  LiveStatCards, KitchenQueuePreview (read-only), RecentCancellations, PeakHourAlert banner.
SalesReportsPage  —  DateRangePicker (preset + custom), SalesSummaryCards, RevenueBarChart, CancellationSection, ExportButtons (PDF/Excel).
StaffManagementPage  —  StaffListTable, Create/Edit/Deactivate modals, StaffDetailDrawer with ShiftLogTable + PerformanceMetrics.
MenuManagementPage  —  CategoryManager, ProductListTable, ProductFormModal (tabs: Basic / Ingredients / Dietary / Recommendations), AvailabilityToggle.
ShiftManagementPage  —  WeeklyScheduleCalendar (staff x day grid), AssignShiftModal, AttendanceLogTable, OvertimeSummaryCard.
SalaryPage  —  SalaryProfilesTable, EditSalaryProfileModal, MonthlyPayrollSheet, SalaryAdjustmentModal (logged), MarkAsPaidModal, SalarySlipPreview.
ExpenseTrackingPage  —  ExpenseForm (date/category/amount/receipt upload), ExpenseListTable, BudgetVsActualChart, ExportButton.
FeedbackPage  —  RatingOverviewCards, FeedbackTable, LowRatedItemsAlert (3+ consecutive), GoogleReviewFunnelChart.
BroadcastPage  —  BroadcastComposer (message + image + audience filter), ScheduleSelector, BroadcastHistoryTable, StatsCard.
QRCodeManagerPage  —  TableQRList, PrintQRButton. Generates unique QR per table for dine-in mobile ordering.

PANEL 5 — CEO
Device: Desktop / Mobile   |   Auth: Password + PIN (2-step) — Read Only
BIDashboardPage  —  GrandTotalCard, RevenueLineChart (daily/weekly/monthly/yearly toggle), RevenueBySourceChart (donut), PeakHoursHeatmap.
BIDashboardPage (cont.)  —  BestSellingItemsTable, ReturningVsNewCustomers doughnut, UpsellConversionCard, PayrollVsRevenueCard.
InvoiceDownloadPage  —  DateRangeSelector (9 presets), CustomDatePicker, CustomHourPicker (specific date + time range), InvoicePreview, DownloadPDF/Excel.
SalaryHistoryPage  —  PayrollSummaryCard, StaffPayrollTable, FlaggedAdjustmentsList (with reason), PayrollTrendChart (year-on-year).
CancellationsRefundsPage  —  RefundSummaryCards, CancellationReasonsChart, RefundTransactionTable. Read-only.
DemandForecastPage  —  WeeklyForecastCard, ForecastVsActualChart, ForecastAccuracyBadge. Requires 4 weeks of data to activate.
ExpensePnLPage  —  RevenueSummaryCard, ExpenseSummaryCard, NetProfitCard — Revenue minus Expenses equals Profit as gauge chart.
BranchSwitcher  —  Phase 2 — toggle between All Branches / Branch A / Branch B for consolidated or per-branch view.

PANEL 6 — DELIVERY BOY
Device: Mobile (React Native)   |   Auth: Staff ID + Password
OrderQueueScreen  —  Active delivery orders only. OrderCard: order#, customer name, address, COD amount. Pull-to-refresh.
OrderDetailScreen  —  CustomerInfoCard (name + tap-to-WhatsApp + address), ItemsListCard, TotalAmountCard (COD to collect).
OpenInMapsButton  —  Launches Google Maps with delivery address pre-filled. One tap navigation.
OutForDeliveryButton  —  Status update. Triggers WhatsApp notification to customer with delivery boy name + ETA.
MarkDeliveredButton  —  Status update. Triggers Delivered WhatsApp + feedback request. Updates Receptionist panel.
OfflineIndicator  —  Shows banner when no internet. Orders cached locally until reconnected.

N8n admin panel restricted to developer/IT admin only
